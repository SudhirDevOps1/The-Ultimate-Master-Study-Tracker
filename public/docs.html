🚀 Getting Started

What is FlowTrack?

FlowTrack is a complete study management system built for students who want to:

  • Track exact time spent studying (not estimated)
  • Automatically detect and prevent studying while distracted
  • Analyze study patterns and optimize productivity
  • Get personalized AI coaching based on real data
  • Maintain daily streaks and achieve study goals

First-Time Setup

✓ 5-minute setup:
  1. Create your first Subject (Math, English, etc.)
  2. Plan study sessions on the Calendar
  3. Start a session with the ⏱️ Timer
  4. Go to Settings and enable Strict Focus Mode
  5. Configure your 🤖 AI Assistant (optional)

🎯 Strict Focus Mode

RECOMMENDED

What It Does

Prevents inflated study time. If you're not actively studying (no clicks, keys, touches) for 10 minutes, your session automatically pauses.

Enable Strict Focus Mode

Settings → Timer Options → Strict Focus Mode (Toggle ON)

How It Works

Timeline of a Study Session:
├─ 0:00  → Start studying (activity detected)
├─ 5:00  → Still active (keeps running)
├─ 8:00  → Step away for coffee (no activity)
├─ 10:00 ← AUTO-PAUSE (notification sent)
│         ↓ actualSeconds frozen
│         ↓ Status changes to "paused"
└─ Return to desk → Resume session

Detected Activities

  • ✓ Mouse clicks and movement
  • ✓ Keyboard input (all keys)
  • ✓ Touchscreen taps and swipes
  • ✓ Page scrolling

Notifications

When auto-paused, you get:
  • Desktop notification: "FlowTrack - Auto Pause"
  • Message: "No activity for 10 minutes. Session paused."
  • Session appears in "Paused" section

🔧 Troubleshooting

Sessions Not Saving

Problem: Sessions appear but disappear after reload

Solutions:

  • Check IndexedDB enabled in browser settings
  • Ensure 5-20 MB free storage
  • Clear browser cache and reload
  • Try different browser
  • Export data before clearing storage

Auto-Pause Not Working

Problem: Session doesn't pause after 10 minutes

Solutions:

  • Verify Strict Focus Mode is ON (Settings)
  • Check session status is "in_progress"
  • Allow desktop notifications
  • Refresh page and try again
  • Check browser console for errors (F12)

AI Assistant Errors

Problem: Connection test fails or errors

Solutions:

  • Verify API key is correct (no extra quotes)
  • Check internet connection
  • Confirm provider website is online
  • Check token/credit balance if paid
  • Try different provider

Performance Issues

Problem: App is slow or lagging

Solutions:

  • Export data and clear storage
  • Close unnecessary browser tabs
  • Disable animations (Settings)
  • Update browser to latest version
  • Try in Incognito mode
