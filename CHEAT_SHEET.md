## ⏱️ Timer Quick Start

```
1. Select Subject (e.g., "Math")
2. Choose Duration:
   • 25m (Pomodoro) - Quick focus
   • 45m (Deep work) - Complex tasks
   • 90m (Flow) - Deep immersion
   • Custom - Your choice
3. Click "Start Focus Session"
4. Study for the duration
5. Session auto-saves
```

**Pomodoro Pattern:**
```
25m Study → 5m Break → 25m Study → 5m Break
→ 25m Study → 5m Break → 25m Study → 15m Long Break
= 4 Pomodoros = ~2 hours productive work
```

## 🎮 Gamification System

### XP Calculation
```
Base XP = Session Duration (minutes)

Multipliers:
• Standard: 1.0x (normal focus)
• High Focus (70%+): 1.5x
• Distraction-Free: 2.0x

Example:
25-minute session with 80% focus = 25 × 1.5 = 37.5 XP
```

### Leveling

Level 3: 250-450 XP

## 🚀 Productivity Tips

### The Pomodoro Method
```
• 25m intense focus (Pomodoro)
• 5m short break (rest, stretch)
• 45m = 1 "cycle" (3 Pomodoros + 1 long break)
• Aim for 6-8 cycles per day = 6-8 hours work
```

### Best Practices
```
✅ DO:
- Study during peak hours (check analytics)
- Take breaks between sessions
- Eliminate distractions
- Track every session
- Review progress weekly
- Set achievable goals

❌ DON'T:
- Multi-task during focus sessions
- Skip breaks
- Set unrealistic goals
- Ignore analytics patterns
- Study when exhausted
- Expect immediate results
```

### Focus Techniques
```
1. Pomodoro (25m) - Classic, proven
2. Deep Work (45-90m) - For complex tasks
3. Time Blocking (hourly) - Schedule-based
4. Task Batching - Group similar work
5. The Two-Minute Rule - Start with 2m
```

## 📊 Metrics Explained

### Time Metrics
```
Active Time = Productive + Distracting + Neutral time
Idle Time = Computer is idle or screensaver

Total Time = Active + Idle
```

### Performance Metrics
```
Streak = Consecutive days meeting daily goal
Focus Score = % of time on productive apps
Productivity = Total study time ÷ Calendar days
Consistency = Days studied ÷ Total days
```

### Progress Metrics
```
Daily Progress = Hours Today ÷ Daily Goal
Weekly Progress = Hours This Week ÷ Weekly Goal
Monthly Progress = Hours This Month ÷ Target
```

## 💡 Pro Tips

1. **Consistency > Intensity**
   - 1h every day > 7h once a week

2. **Review Regularly**
   - Weekly: Check analytics, adjust goals
   - Monthly: Analyze trends, plan next month

3. **Track Everything**
   - Every session counts
   - Small sessions add up

4. **Use Subjects**
   - Better organization = better focus
   - Easier to identify weak areas

5. **Monitor Focus Score**
   - High score = productive studying
   - Low score = too many distractions

6. **Build Streaks**
   - Aim for consistent daily studying
   - Even short sessions maintain streak

7. **Analyze Patterns**
   - What times are you most productive?
   - Which subjects take most time?
   - When do you have most distractions?

8. **Celebrate Progress**
   - Unlock achievements
   - Level up regularly
   - Hit streaks and goals
